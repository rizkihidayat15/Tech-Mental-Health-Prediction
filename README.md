===================== Cara Pertama | Menggunakan VS Code =====================

Ekstrak semua file didalam zip ini.

2. Buka folder "Tech Mental Health Prediction" di VSCode

3. Didalam VSCode, buka buka terminal baru dengan tombol shortcut Ctrl + `.

4. Di terminal, ketik cd backend

5. Lalu ketik .\venv\Scripts\activate.ps1

6. Jika error, ketik prompt ini Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned dan ketik kembali prompt di step 5.

7. Setelah ada (venv) PS... muncul, tandanya sudah berhasil masuk ke virtual environment python.

8. Setelah masuk ke virtual environment, ketik python app.py

9. Jika berhasil, akan ada pesan Running on http://127.0.0.1:5000 dan backend siap digunakan.

10. Untuk menjalankan website nya diperlukan extensions bernama "Live Server".

11. Jika sudah ada ekstensi Live Server, selanjutnya adalah cari "index.html" di folder "ui" lalu klik "Go Live" di pojok kanan bawah window VSCode.

12. Selamat! Anda dapat menikmati hasil proyek kami.


===================== Cara Kedua | Menggunakan XAMPP / Laragon =====================

Ekstrak semua file didalam zip ini.

2. Pindahkan folder "ui" ke htdocs (jika XAMPP) atau www (jika Laragon).

3. Buka CMD dan arahkan ke folder "backend".

4. Lalu jalankan prompt venv/Scripts/activate

5. Jika error, jalankan prompt venv/Scripts/activate.bat

6. Jika ada pesan Running on http://127.0.0.1:5000 maka backend siap digunakan.

7. Aktifkan XAMPP / Laragon.

8. Lalu buka folder "ui" di browser dengan localhost

9. Selamat! Anda dapat menikmati hasil proyek kami.
